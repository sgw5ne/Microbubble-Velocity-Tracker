# Microbubble-Velocity-Tracker

Given an ultrasound video, this code will attempt to detect bubbles, measure their velocities, and plot a distribution of the velocities

## Setup

First clone this repository onto your device. In order to do this, your device will need to have github installed on it. Navigate to a directory of your choosing and run the command `git clone [insert link here]`. This will copy all of the code into your chosen folder.

Before running this program, you will first want to ensure that python and all of the required packages are installed on your machine. I would recommend starting a virtual environment before installing the required packages, which can be done using the command `python -m venv .venv`. After starting the virtual environment, you can install the required packages using `pip install -r requirements.txt`. Once all of the requirements are installed, you can run the code itself using `python main.py`.
